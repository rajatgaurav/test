# Petro Power Engineering Services Website

Modern Next.js 14 website with TypeScript and Tailwind CSS.

## Setup

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Build

```bash
npm run build
npm start
npm run build && next export -o out
```

## Features
- Responsive design
- TypeScript
- Tailwind CSS
- SEO optimized
- Fast performance
